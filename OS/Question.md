# Top 50 Operating System (OS) Interview Questions for Freshers

## 1. What is an Operating System?

An Operating System (OS) is system software that acts as an interface between the user and computer hardware.

---

## 2. What are the main functions of an OS?

* Process management
* Memory management
* File management
* Device management
* Security management

---

## 3. What is a process?

A process is a program in execution.

---

## 4. What is a thread?

A thread is the smallest unit of CPU execution within a process.

---

## 5. What is the difference between a process and a thread?

| Process          | Thread        |
| ---------------- | ------------- |
| Heavyweight      | Lightweight   |
| Own memory space | Shared memory |
| More overhead    | Less overhead |

---

## 6. What is multitasking?

Running multiple tasks simultaneously.

---

## 7. What is multiprogramming?

Keeping multiple programs in memory to maximize CPU utilization.

---

## 8. What is multiprocessing?

Using multiple CPUs to execute processes.

---

## 9. What is multithreading?

Executing multiple threads within a process.

---

## 10. What is a kernel?

The kernel is the core component of an operating system that manages hardware and resources.

---

## 11. What is system call?

A system call allows a user program to request services from the operating system.

---

## 12. What is context switching?

Saving one process state and loading another process state.

---

## 13. What is a CPU scheduler?

It selects the next process to execute.

---

## 14. What is CPU scheduling?

The process of selecting which process gets CPU time.

---

## 15. What is FCFS scheduling?

First Come First Serve scheduling.

---

## 16. What is SJF scheduling?

Shortest Job First scheduling.

---

## 17. What is Round Robin scheduling?

Each process gets a fixed time slice called a time quantum.

---

## 18. What is Priority Scheduling?

CPU is allocated to the process with the highest priority.

---

## 19. What is preemptive scheduling?

The CPU can be taken from a running process.

---

## 20. What is non-preemptive scheduling?

A process keeps the CPU until it finishes or blocks.

---

## 21. What is deadlock?

A situation where processes wait indefinitely for resources.

---

## 22. What are the necessary conditions for deadlock?

1. Mutual Exclusion
2. Hold and Wait
3. No Preemption
4. Circular Wait

---

## 23. What is starvation?

A process never gets CPU resources.

---

## 24. What is aging?

Increasing process priority over time to avoid starvation.

---

## 25. What is a semaphore?

A synchronization tool used to control access to shared resources.

---

## 26. What is mutex?

A locking mechanism that allows only one process or thread to access a resource.

---

## 27. What is synchronization?

Coordinating multiple processes or threads.

---

## 28. What is critical section?

The code segment accessing shared resources.

---

## 29. What is paging?

Memory management technique that divides memory into pages.

---

## 30. What is segmentation?

Memory management technique that divides memory into logical segments.

---

## 31. What is virtual memory?

A technique that allows execution of programs larger than physical memory.

---

## 32. What is demand paging?

Pages are loaded only when required.

---

## 33. What is page fault?

Occurs when a required page is not found in memory.

---

## 34. What is thrashing?

Excessive paging causing low CPU utilization.

---

## 35. What is fragmentation?

Wastage of memory space.

---

## 36. What are the types of fragmentation?

* Internal fragmentation
* External fragmentation

---

## 37. What is swapping?

Moving processes between RAM and disk.

---

## 38. What is cache memory?

High-speed memory between CPU and RAM.

---

## 39. What is a file system?

A method for organizing and storing files.

---

## 40. What is a directory?

A collection of files and folders.

---

## 41. What is spooling?

Storing jobs temporarily in a buffer for device processing.

---

## 42. What is a bootloader?

A program that loads the operating system during startup.

---

## 43. What is booting?

The process of starting the computer.

---

## 44. What are the types of booting?

* Cold Boot
* Warm Boot

---

## 45. What is RAID?

A technology that combines multiple disks for performance or reliability.

---

## 46. What is IPC?

Inter-Process Communication used for communication between processes.

---

## 47. What are IPC methods?

* Pipes
* Message Queues
* Shared Memory
* Sockets
* Semaphores

---

## 48. What is a real-time operating system (RTOS)?

An OS that guarantees response within a specific time.

---

## 49. What is the difference between user mode and kernel mode?

| User Mode             | Kernel Mode          |
| --------------------- | -------------------- |
| Limited access        | Full hardware access |
| Less privilege        | Highest privilege    |
| User applications run | OS kernel runs       |

---

## 50. What is the difference between 32-bit and 64-bit OS?

| 32-bit OS               | 64-bit OS            |
| ----------------------- | -------------------- |
| Supports up to 4 GB RAM | Supports large RAM   |
| Smaller address space   | Larger address space |
| Lower performance       | Better performance   |
